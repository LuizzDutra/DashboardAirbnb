# How to setup the enviroment


**This Project was made with Vite**<br>
It uses React with Typescript

## Running in development
_(Make sure your working directory is the ./frontend folder, the folder of this README)_

To run the enviroment in development do the following<br>

1. Install all dependencies with:
    ```bash
    npm install
    ```
2. Run in development with:
    ```bash
    npm run dev
    ```

3. Make sure the server is running on port 5173 and access it through localhost
