This project is a Dashboard made with React + Django.

The dataset utilized was [New York AirBnB Open Data 2024](https://www.kaggle.com/datasets/vrindakallu/new-york-dataset)<br>
The exploration and cleaning of the data to extract the CSVs was done on a google colab [Notebook](https://colab.research.google.com/drive/1sCRfCEH2D5QON_hs3Yp0ul6PBFnyT3iE?usp=sharing)

To run the project in your enviroment follow the respective READMEs of the front and back end.
